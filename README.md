# 🍴 Food Menu Application (Beginner .NET Project)

A simple **.NET beginner project** built to explore the fundamentals of web development using .NET.  
This project is designed as a **Food Menu Application**, where users can **browse menu items, view item details, and search for specific items**.  

---

## 🚀 Features
- 📜 **View Menu** – See a list of all available food items  
- 🔍 **Search** – Search for food items by name  
- ℹ️ **View Details** – Click on a food item to see more details  

---

## 🛠️ Technologies Used
- **.NET (ASP.NET Core MVC)**  
- **C#**  
- **Razor Pages / MVC Views**  
- **Rider IDE / Visual Studio**  

---

## 📂 Project Structure
FoodMenuWebApplication/
│-- Controllers/ → Handles requests and app logic
│-- Models/ → Data models (FoodItem, etc.)
│-- Views/ → UI (Razor views for displaying menu, search, details)
│-- wwwroot/ → Static files (CSS, JS, images)
│-- appsettings.json → Configuration file
